# ローカル連携のセキュリティとプライバシー

## 対象範囲

DopaRush Mixer は Windows のローカル音声セッションを操作するデスクトップアプリです。Vivaldi タブ単位の音量制御は、同梱の開発者モード用拡張機能とローカル HTTP ブリッジを利用する任意機能です。

この連携は個人利用を想定したカジュアルな実装であり、認証付きの安全なプロセス間通信として設計されていません。共有端末、信頼できないローカルソフトウェアが動作する環境、または強い隔離が必要な環境では使用しないでください。

## 扱う情報

- ミキサー本体は、再生中アプリのプロセス名、PID、オーディオセッション識別子、出力デバイス識別子をローカルで読み取ります。
- 出力デバイスの利用履歴は、最大12件のデバイス識別子として `%LocalAppData%\DopaRushMixer\output-device-history.json` に保存されます。
- Vivaldi 連携では、登録したタブのタイトル、favicon URL、音量・ミュート状態と、音声を出している未登録タブのタイトルをローカルブリッジへ送ります。
- タブ音声は拡張機能内の Web Audio API でゲイン調整に使います。このソースコードは音声データを保存せず、インターネット上のサーバーへ送信もしません。

## ローカル HTTP ブリッジの制約

ブリッジは `http://127.0.0.1:32145/` だけで待ち受けます。LAN やインターネットから直接接続できるようには待ち受けません。

一方で、次の意図的な制約があります。

- 通信は TLS で暗号化されず、認証トークンも使いません。
- CORS は全 Origin を許可します。
- 同じ端末上で動く別プロセスは、ブリッジへ偽のタブ状態を送ったり、保留中の拡張機能コマンドを読み出して消費したりできます。

したがって、このブリッジは悪意あるローカルプロセスから情報や操作を守る境界ではありません。Vivaldi 連携を有効にするかどうかは、この前提を理解した利用者自身の判断に委ねられます。

## 拡張機能の権限

拡張機能は `tabs`、`tabCapture`、`offscreen` の権限と、上記 localhost へのホスト権限を要求します。Web Store 配布用の拡張機能ではなく、リポジトリから利用者が開発者モードで読み込むことを想定しています。
