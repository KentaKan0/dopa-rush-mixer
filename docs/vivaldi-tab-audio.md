# Vivaldi タブ単位の音量制御

Windows のオーディオミキサーは Vivaldi のタブを別々に識別できないため、同梱の Vivaldi 拡張機能で音声を取り込み、タブごとにゲインを適用する。

この機能はローカル利用向けのカジュアルな実装である。導入前に、[ローカル連携のセキュリティとプライバシー](security-and-privacy.md) を確認する。

## 導入

1. Vivaldi で `vivaldi://extensions` を開く。
2. 「デベロッパーモード」を有効にする。
3. 「パッケージ化されていない拡張機能を読み込む」を選び、`vivaldi-extension/` フォルダーを指定する。
4. 音量ミキサーを起動する。
5. 個別制御したいタブを表示し、拡張機能のツールバーアイコンをクリックして登録する。

登録済みタブが音量ミキサーの「Vivaldi タブ」欄に表示される。タブを閉じた後は、拡張機能を再読み込みするかブラウザを再起動して登録を整理する。

音声を出している未登録タブは自動検出される。拡張機能アイコンのバッジに件数が表示され、ミキサーの「Vivaldi で音声を検出」欄にも表示される。対象タブを開き、拡張機能アイコンをクリックして個別制御を有効化する。

「Vivaldi タブ音量」欄の上部にある VIVALDI MASTER は、登録済み Vivaldi タブすべてへ適用される音量とミュートである。個別タブの設定はその下で調整できる。未登録タブの検出欄は折りたたみ可能で、通常はヘッダーだけの高さになる。登録済みタブはページ遷移時のタイトル変更とファビコン変更にも追従する。ファビコンを取得できないページでは、Vivaldi のアイコンを表示する。

Windows が vivaldi のオーディオセッションを公開している間は、VIVALDI MASTER がそのセッションにも直接連動し、通常のアプリ一覧には重複表示しない。音声セッションがない場合でも VIVALDI MASTER は待機表示のまま残り、登録済みタブの一括制御を行える。

登録済みタブの左端へマウスを移動すると、解除用の × ボタンが現れる。解除すると、そのタブの個別音量制御と音声キャプチャを停止する。タブごとのリアルタイムメーターは操作の滑らかさを優先して表示しない。

拡張機能アイコンを押す代わりに、対象タブをアクティブにして `Ctrl+Shift+Y` を押しても登録できる。ショートカットが競合する場合は `vivaldi://extensions/shortcuts` で変更する。

## 制約

- Chromium の `tabCapture` はユーザー操作で開始する必要があるため、タブごとに拡張機能アイコンを一度クリックする必要がある。
- ミキサーアプリの操作やローカル通信は Chromium のユーザー操作とは見なされないため、ミキサー側からタブキャプチャを開始することはできない。
- 保護されたページや DRM コンテンツではタブ音声の取得に失敗する場合がある。
- 拡張機能を再読み込み・無効化すると登録済みタブの個別制御は停止する。
