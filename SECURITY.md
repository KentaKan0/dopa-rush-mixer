# セキュリティ方針

## 対象範囲

Vivaldi 連携の localhost ブリッジは、個人利用を想定した認証なしのカジュアルな実装です。既知の制約と扱う情報は [ローカル連携のセキュリティとプライバシー](docs/security-and-privacy.md) を参照してください。

## 報告時の注意

- 資格情報、個人情報、実際のタブ名、ローカルログ、音声データを公開 Issue に投稿しないでください。
- GitHub の **Report a vulnerability** が表示される場合は、そちらを優先してください。
- 非公開の報告経路が利用できない場合は、機微な再現情報を含む報告を公開 Issue に投稿しないでください。

外部 Pull Request は受け付けません。セキュリティに関する公開前提の変更も、所有者の判断で管理します。
